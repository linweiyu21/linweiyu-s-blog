kubectl 命令工具

$ kubectl [command] [TYPE] [NAME] [flags]

TYPE: 资源对象类型, 区分大小写, 能以单数, 复数或者简写形式表示

获取多个 Pod 的信息

$ kubectl get pods pod1 pod2

获取多种对象的信息

$ kubectl get pod/pod1 rc/rc1

同时应用多个 YAML 文件

$ kubectl apply -f pod1.yaml -f pod2.yaml

使用目录下的所有文件定义

$ kubectl create -f <directory>

删除所有 Pod

$ kubectl delete pods --all

在 Pod 中执行命令 (默认使用 Pod 中第一个容器)

$ kubectl exec <pod-name> <command>

在 Pod 指定容器中执行命令

$ kubectl exec <pod-name> -c <container-name> <command>

在 Pod 指定容器中运行 bash

$ kubectl exec -ti <pod-name> -c <container-name> /bin/bash

在线编辑运行中的资源对象

$ kubectl edit deploy nginx

将 Pod 的开放端口映射到本地

$ kubectl port-forward --address 0.0.0.0 pod/nginx-2832834829-2323 8888:80

在 Pod 和本地之间复制文件

$ kubectl cp nginx-snnsowe983-wb23sd:/etc/fstab /tmp

检查可用的 API 资源类型列表

$ kubectl api-resourcecs

查看已安装的插件列表

$ kubectl plugin list
