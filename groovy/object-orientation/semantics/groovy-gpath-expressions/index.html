this.class

  • property accessor
  • equivalent to this.getClass() in Java
  • yield a Class object

this.class.methods

  • property accessor
  • equivalent to this.getClass().getMethods()
  • yield ++an array++ of Method objects

this.class.methods.name

  • apply a property accessor on each element of an array
  • produce ++a list++ of the results

this.class.methods.name.grep(...)

  • call method grep on each element of the list
  • produce ++a list++ of the results
